# Agentic AI Watch Output

This folder stores timestamped markdown reports generated by the Agentic AI watch script.

## Files

- `agentic_ai_watch_YYYY-MM-DD_HH-MM-SS.md`: A point-in-time report of matching official articles and release notes.

## Generator

- Script: `tools/one-off/run_agentic_ai_watch.py`
- Continuous runner: `tools/one-off/run_agentic_ai_watch_daily.ps1`
- Windows Task Scheduler helper: `tools/one-off/register_agentic_ai_watch_task.ps1`
- Digest generator: `tools/one-off/generate_agentic_ai_watch_digest.py`
- Watch + digest daily runner: `tools/one-off/run_agentic_ai_watch_and_digest_daily.ps1`
- Watch + digest task helper: `tools/one-off/register_agentic_ai_watch_and_digest_task.ps1`

## Notes

- Reports are generated from official source pages and feeds only.
- The script writes a new file on every run so historical snapshots are preserved.
- The digest generator consolidates report content, removes duplicate URLs, and lists the latest published items first.
