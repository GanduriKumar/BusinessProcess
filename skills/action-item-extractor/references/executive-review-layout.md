# Executive Review Layout

Use this guidance when converting extracted meeting outcomes into an executive review deck or slide summary.

## Recommended slide spine

1. Executive takeaway
2. What leaders asked
3. Question and intent heatmap or table
4. Confirmed action items
5. Risks, blockers, and unresolved asks
6. Recommendation / next step

## Most useful table format

Use a compact table with these columns:

- Question / input heard
- Underlying intent
- Suggested approach / action item
- Owner
- Due date
- Status

## Suggested visual treatment

- White or light-gray background
- Strong single-line slide headline
- One accent color family for highlights, with amber/red only for risks or blockers
- Use cards or a heatmap-style table for grouped questions
- Keep dense transcript text off the slide; summarize the intent instead

## Executive-ready formatting

- Headline should answer: what matters from this meeting?
- Limit the main table to the highest-signal questions
- Group similar questions rather than showing transcript chronology
- Use short action-oriented wording, not verbatim transcript language
- End with explicit asks, owners, and next review point
