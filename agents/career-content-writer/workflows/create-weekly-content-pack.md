# Weekly Content Pack Workflow

1. Read the latest `career_content_plan_*.html` file unless a specific plan file is provided.
2. Resolve the selected week and post topic number.
3. Extract:
   - week goal
   - selected daily post topic
   - LinkedIn article topic
   - Medium article topic
   - CTA guidance
4. Generate:
   - LinkedIn post optimized for clarity, response, and executive resonance
   - LinkedIn article optimized for authority and follow/subscription behavior
   - Medium article optimized for evergreen search and external sharing
5. Run a Potato Mode review pass:
   - remove generic language
   - harden the business point of view
   - keep the writing human
6. Write both Markdown and HTML outputs under `docs/output/personal/posts`
