# Positioning Topics Workflow

## Goal
Generate a practical backlog of personal-brand topics and monetizable offers from the source assessment.

## Steps
1. Read the source assessment HTML or markdown.
2. Read the recent VP/SVP-level job market benchmark.
3. Extract recurring themes such as context engineering, governance, observability, advisory positioning, enterprise agentic AI, commercialization, operating model, and executive influence.
3. Convert those themes into:
   - post topics
   - article or downloadable asset topics
   - workshop and advisory offers
4. Save the generated markdown under `docs/output/personal/`.

## Validation
- The output should be specific enough to write from.
- Offers should map to a buyer, problem, and deliverable.
- Themes should match the source assessment rather than drifting into generic AI content.
