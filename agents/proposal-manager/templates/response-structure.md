# Proposal Response Structure

## Direct Response

State the answer in one or two sentences.

## Delivery Approach

Explain how the requirement will be met in practice.

## Evidence

Add proof points such as relevant delivery experience, controls, metrics, assets, or accelerators.

## Governance / Risk Control

State how quality, risk, compliance, or service assurance will be handled.

## Customer Benefit

Close with the practical value to the customer, without overclaiming outcomes.
