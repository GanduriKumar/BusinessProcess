# Executive Deck Workflow

## Objective

Turn analysis or raw content into a concise executive presentation.

## Steps

1. Define the decision
- Identify what the deck needs the audience to understand, approve, or do.
- Write the one-sentence takeaway for the full deck.

2. Build the slide spine
- Cover
- Executive summary
- Current state / problem
- Analysis or evidence
- Options or implications
- Recommendation
- Next steps

3. Draft slide-level messages
- Give each slide one headline takeaway.
- Choose table, chart, card, or narrative based on the information type.

4. Build or revise the deck
- Reuse existing scripts when repeatable generation helps.
- Keep visuals consistent and readable.

5. Tighten for executives
- Remove repeated text.
- Reduce dense slides.
- Ensure the first two slides answer the main question.

## Final check

- One message per slide
- Numbers sourced
- Clear ask or implication at the end
